## [0.0.1] - 2024-12-08

### Added

- Initial version
- Support reading JSON file from a local filesystem
- Support reading JSON content from `stdin`
- Provide help via `komb --help`
- Return proper exit shell code
- Support bb (native GraalVM) and nbb (Node.js)
